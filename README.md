#Tic-Tac-Toe Android App

A simple Tic-Tac-Toe game for Android built with Java and Android Studio.  
Two players can play locally, take turns, and see the score. The app handles wins, draws, and allows restarting the match.



Features
- Two-player local gameplay  
- Win, draw, and score tracking  
- Restart matches at any time  
- Full-screen immersive experience  
- Simple and intuitive UI
